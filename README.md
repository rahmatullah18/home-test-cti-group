## Installation

Clone project menggunakan git

```bash
  git clone https://github.com/rahmatullah18/home-test-cti-group.git
  cd home-test-cti-group
```

Install package

```bash
 npm install

 atau

 yarn install
```

Jalankan program

```bash
  npm run start

  atau

  yarn start
```
